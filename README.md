# Cache_Cheatsheet_Lite

Лёгкий демонстрационный проект ASP.NET Core, показывающий, как реализовать собственное кэширование с использованием:

- IDistributedCache
- Redis
- IAsyncResourceFilter
- Пользовательского атрибута Cache
- Абсолютного и скользящего истечения
- Чёткой архитектурной изоляции

Проект создан как личный справочник по кэшированию, а не как продакшн‑решение.


## 🚀 Технологии

- ASP.NET Core
- Redis (StackExchange.Redis)
- IDistributedCache
- Кастомные Resource Filters
